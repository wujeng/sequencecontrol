Setup Controlled System

Controlled Systems have two part: Pneumatic Actuator and Electrically controlled devices.

Setup pneumatic control system:¡G

  1. Select pneumatic actuator and drag it to a appropriate position.
  2. Select a electrical valve to control the pneumatic actuator and drag it to a appropriate position.
  3. Connect pneumatic devices
  4. Pickup the electrical devices to be used from electrical device pool. The instances of the chosen electrical devices will show in electrical device windows.
  5. Setup limit switch; right click on limit switchs to use popup windows "position" function to choose an actuator for the limit switch to be put on.
     

Setup electrically controlled systems¡G

  1. Select a electrically controlled device from system pool and drag it to an appropriate position.
  2. Pickup the electrical devices to be used from electrical device pool.
  3. Put on limit switch.
     

 

If user wants to delete a controlled system, he right clicks on the controlled system to bring up a popup windows and use "delete" function to delete it.
