User Defined Device

There are so many pneumatic devices. It would be tedious to design a device class for every pneumatic device. Therefore, in this program all pneumatic devices are separated into different groups according to their functions. A general purpose device class is designed for every group. With appropriate information new device can be added into the program easily. The configuration file (resources/config.txt) of this program lists all these information to add new devices. User can modify the configuration file to add user defined devices.
