Setup Operation Conditions

In this software, setup operation conditions means to setup the following conditions:

  1. Setup start button for the control circuit.
  2. If user wants to add emergency stop function, he needs to setup a emergency stop button.
  3. Set operation mode.
    There are four operation modes:
    1. Simple motion:
    2. Continuous motion:
    3. Simple/Continuous motion choice: Use a choice switch to choose simple motion or continuous motion during control process.
    4. Repeated: Use a choice switch to switch between repeat and not repeat, or use a counter to set the repeatition number.

     

Default: Simple motion without emergency stop.
