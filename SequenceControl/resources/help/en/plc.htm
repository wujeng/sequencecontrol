PLC Codes

To translate the electrical circuit into PLC codes, user clicks on the "PLC\Codes" function in the pull-down menu. A PLC code windows will open.

In this software, the electrical circuits designed can be translated into PLC codes of all kinds of PLC. There are many kinds of PLC in the world. Every PLC model has its own hardware structure and software format. It is difficulty to define a universal PLC class. However, in this research when we translate the electric circuits into PLC codes, we only use basic instructions (input/output instructions, logic instructions, and timer/counter instructions), which are basic commands of all PLC except that the instruction format might different and memory addresses are different. Based on this understanding, a simple PLC class is defined (see Table 1). With proper information, users can add PLC model into this software. And the software will be able to translate the electrical circuits into the PLC codes of that model.
