Pull-down Menu

File menu: File operation (When the program is used as Java application not as Java applet.)

New

Start a new design.

Open

Open an old circuit design file.

Save

Save the current circuit design into file.

Save as

Save the current circuit design into a different file.

Import

Import a circuit design file into the current design.

Export

Save the circuit in the selected area into a file.

Exit

Exit the program.

Edit menu: Edit circuit

Group Move

Move the circuit in the selected area to other position of the circuit panel.

Group Cut

Cut the circuit in the selected area off.

Group Copy

Copy the circuit in the selected area to the edit board.

Paste

Paste the circuit in the edit board into current circuit.

Group Delete

Delete the circuit in the selected area.

Save Properties

Save system properties into a file.

DataBase menu: Database access

Login

Login to the database.

New

Start a new design in the database.

Open

Open a circuit design from the database.

Save

Save the current design into the database.

Delete

Delete a circuit design in the database.

Save as

Save the current design into the database with a different name.

Import

Import a circuit from the database into the current circuit.

Export

Save the circuit in the selected area into the database.

MultiUser menu: Collaborative design

Connect

Connect to the collaborative server.

grab token

Get the token to dominate the circuit design process.

release token

Release the token so that somebody else can get it.

Disconnect

 Disconnect from the collaborative server.

Synchronize

Get the circuit design in the collaborative server to synchronize with others.

Chat

Chat with others connected to the collaborative server.

Electrics menu: Computer-aided electrical circuit design

Ladder

Open electrical circuit design windows.
