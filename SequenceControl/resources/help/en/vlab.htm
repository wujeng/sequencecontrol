Virtual Pneumatic Laboratory

Virtual Pneumatic Laboratory is a web-based computer-aided pneumatic circuit design software. it has following functions:

  1. Pneumatic circuit computer-aided design:
    The program helps user pick pneumatic devices and connect them to form pneumatic control circuit. All devices are represented by device symbols.
  2. Animatedly simulation:
    All devices in the circuit are simulated all the time.
  3. Central data storage (Server version, see register):
    All users' circuit design data can be stored in a database with the usages of a web server and a database server.
  4. Collaborative design(Collaborative version, see register):
    Working with a collaborative server, this software can let many users design circuits at the same time collaboratively.
  5. Devices provided by user:
    With appropriate pneumatic device symbols, user can edit a configuration file to add new devices into the program.
