Computer Automatic Design

In the pull-down menu of WebLadderCAD by executing the "AutoDesign/Sequence" function, user can open up a motion sequence windows. User edits motion sequence in this windows and asks the program to design electrical circuit automatically according to the motion sequence requirement.

The procedures of automatic design are stated in the following:

  1. Setup the controlled systems:
    Setup pneumatic control circuit or/and electrical controlled devices. They are shown in the pneumatic circuit panel.
  2. Input motion sequence requirement:
    Use computer-aided motion sequence editing to input motion sequence with specific format.
  3. Setup operation conditions:
    Use the menu function "AutoDesign/Condition" of the motion sequence windows to setup up operation conditions.
  4. Execute computer automatic design:
    Use the "AutoDesign/Design" menu function to let the program design electrical control circuit according to the motion sequence requirement.
