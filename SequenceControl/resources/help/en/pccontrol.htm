PC-Based Sequential Control

The control part of the sequential control software is integrated into WebLadderCAD in the "control" menu. The control part of the software is designed to satisfy following requirement:

  1. User can perform local-side sequential control.
  2. User can perform remote-side sequential control.
  3. Control results can be monitored locally or remotely.
  4. Local-side and remote-side use the same program.
Due to noisy industry enviroment, industry personal computer is used as controller for its reliability. The Web-based sequential controller uses central control design. Only remote data acquisition modules are exposed to the noisy enviroment. The controller could be made of regular personal computer.

The "control" menu of Web-LadderCAD is shown here:
Server Enter server mode.
Client Enter client mode.
Hardware Scan/Get for hardware information.
Grab Control Grab the right to control.
Release Control Release control right.
Upload Upload circuit design to server program.
Download Download circuit design from the server program.
Assign Assign input/output connections.
Start Control Start the control action.
Stop Control Stop the control action.

When user wants to perform PC-based sequential control, he clicks on the "Control/Server" function in the pull-down menu to enter server mode. Then, he clicks on "Control/Hardware" function to scan through RS232 port to get the information of NAP7000 remote data acquisition modules which are online. He then executes "Control/Grab Control" to get control right.

Then, he has to assign input/output connections. If the input/output mapping is correct, he finally can click on "Control/Start" menu to start control action.

During the control action, he can click on "Control/Stop" menu to stop the control action anytime.

Web-based Sequential Controller can be used in remote sequential control and monitoring. The server program could be accessed by many client programs at the same time. Whoever wants to perform control action must get control right first. The client program which does not have the control right can only monitor the control result.
