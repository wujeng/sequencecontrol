Pick up device


  1. Switch to appropriate device pool, click on the device symbol you wanted.
    (An instance of the device symbol you choosen will show up at the up-left corner of the circuit panel.)
  2. Drag the device symbol to appropriate position with mouse.
  3. When user want to install limit switch valves on certain actuator device, he just drags the device into the actuator device.
    A little limit switch mark will shown up in the actuator symbol.
    User can drag around the little limit switch mark to change limit switch position or take off limit switch.
  4. If user wants to change the direction of certain pneumatic device, he can right click on the device to use "rotate" function on the popup menu. "rotate" function will rotate the device 90 degree clock-wide.
  5. If user wants to delete certain device, he can right click on the device to use "delete" function on the popup menu.
