Pull-down Menu of WebLadderCAD

Edit menu: Edit of electrical circuit.

Delete Delete the electrical circuit in the selected block.
Cut Cut the electrical circuit in the selected block into the board.
Paste Paste the electrical circuit in the board into the position of the cursor.
Copy Copy the electrical circuit in the selected block into the board.
Overwrite Switch the edit mode (Insert or Overwrite).

Simulation menu: Animated simulation
Analysis Analyze the electrical circuit.
Simulation Start the animated simulation of the controlled system controlled by the electrical circuit.
Stop Stop the simulation.

Control menu: Control with I/O modules
Server Let the program run as a server.
Client Let the program run as a client.
Hardware Get/Set the hardware configuration.
Grab Control Grab the control right.
Release Control Release the control right.
Upload Upload the local circuit design to the remote server.
Download Download the circuit design in the server to this client.
Assign Assign input/output connections.
Start Control Start the control action.
Stop Control Stop the control action.

AutoDesign menu: Automatic design of electrical circuit according to the motion sequence required.
Sequence Open motion sequence windows.

PLC menu: PLC codes of different PLC models.
Codes Open a PLC windows to show the PLC prgram translated from the electrical circuit.
