Assign Input/Output Mapping

Assignning input/output connections means three things:

  1. Connect input signals, like limit switch signal, push button signal, ..., to remote digit input data module.
  2. Connect output signals, like electrical valve line, motor line, alarm line,..., to remote digit output data module.
  3. Mark these connections in the electrical devices in the electrical device windows of WebLadderCAD.
    Right click on the input/output electrical device to bring up popup menu. Use the "connection" function to assign connection.

    Shown as figure above, right click on LS1 device, execute "Connection" function. A connection windows will be popuped to let user assign connection.
