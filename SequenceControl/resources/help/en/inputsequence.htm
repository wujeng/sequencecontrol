Input Motion Sequence

There are many methods to express motion sequence. This program uses motion symbol method to describe the motion sequence.

In this software, motion sequence is formed by motion symbols and branch marks. To enter branch mark in the cursor position, user clicks on the mark button in the tool bar. To enter motion symbol in the cursor position, user clicks on the "add" button. Before pressing "add" button, user has to select respective controlled system, motion direction, and end switch.

Block Edit:
User can use mouse to drag out a selected block in the motion sequence panel. Then, he can operate (move, delete, cut, ...) on the sequence block by using popup windows.
